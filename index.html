Install Planix to any disk/storage device (SSD, HDD, USB, SD, etc.) using the following command: 
 xzcat planix-1.0-aarch64.img.xz | sudo dd of=/dev/[target device] bs=4M status=progress conv=fsync

Replace [target device] with the ID of the target device. e.g. sda
WARNING: To avoid selecting the wrong device, run lsblk. Do NOT select the device mounted at /
